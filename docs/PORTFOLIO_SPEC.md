# Portfolio Specification

The portfolio uses a black base, white type, neutral grays, thin editorial rules, large local system typography, and one coherent motion language. The primary audience is technical recruiters and engineering teams evaluating Kendall's fullstack, AI, QA automation, and DevOps work.

The first viewport identifies Kendall immediately. Projects form the main visual narrative: a sticky horizontal sequence on wide screens and a readable vertical list on smaller screens or reduced motion. Experience and education share an SVG timeline. Contact is a quieter close.

The WebGL layer is progressive enhancement. It must never block reading, navigation, or interaction.
